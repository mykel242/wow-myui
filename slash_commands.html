üìà Meter Scaling Explained

  • Auto-scaling: Meters automatically scale based on your previous combat peaks with 40% headroom
  • Manual scaling: Use dpsmax/hpsmax to set fixed maximum values
  • Reset: Use reset commands to return to auto-scaling behavior
  • Persistence: Manual scaling settings are saved between game sessions

üîç Debug Output

When debug mode is enabled (/myui debug), you'll see:

  • Combat event parsing details
  • Peak value tracking updates
  • Rolling average calculations
  • Meter scaling decisions
  • Automatic sample data every 5 seconds during combat

üìù Important Notes

  • All window positions are automatically saved when you drag them
  • Meter scaling preferences persist between game sessions
  • Debug mode setting is saved in your addon data
  • Combat testing commands are for development/testing only
  • Meters show "(auto)" or "(manual)" status in the bottom indicator
  • Rolling averages are calculated over 5-second windows for smoother display
